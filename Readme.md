# 广告过滤规则自动更新

个人使用Python脚本对网络规则进行去重，并生成新的规则文件。
可以添加多个数据来源，并去重。

## 功能

- 从多个URL获取广告过滤规则
- 自动去除重复规则
- 生成包含统计信息的输出文件
- 每天自动更新
